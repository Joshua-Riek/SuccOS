# SuccOS
 A hobby Operating System developed from scratch using i8086 assembly.

## Building source
Using both Nasm & GCC Cross-Compiler:
```sh
make
```

Using only Nasm:
```sh
nasm -f bin src/kernel.asm -o obj/kernel.bin
```
